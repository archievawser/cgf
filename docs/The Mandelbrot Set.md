2D visualizations of the Mandelbrot treat each image coordinate as coordinates of the complex plane / complex numbers. In iterating the following formula,
$$f_c(z) = z^2 + c$$
where initially $z=0$  (idk if that is always true) and $c$ is the complex number being tested, the set can be approximated. Any complex number which this function does *not* diverge to infinity for after an infinite number of iterations is within the Mandelbrot set.
